# PESTEL 宏观环境分析

## 模型概览

```
┌─────────────────────────────────────────────────────────┐
│                    PESTEL 分析框架                        │
├──────────┬──────────┬──────────┬──────────┬──────────┬───┤
│    P     │    E     │    S     │    T     │    E     │ L │
│Political│ Economic │  Social  │Technology│Environ. │Legal│
│  政治   │   经济   │   社会   │   技术   │   环境   │法律│
│  ─────  │  ─────   │  ─────   │  ─────   │  ─────   │────│
│ 政策    │ 经济增长  │ 人口结构  │ 技术创新  │ 气候变化  │ 监管 │
│ 稳定性  │ 利率汇率  │ 生活方式  │ 数字化   │ 环保意识  │ 合规 │
│ 贸易    │ 通胀失业  │ 价值观   │ 自动化   │ 可持续   │ 知产 │
└──────────┴──────────┴──────────┴──────────┴──────────┴───┘
```

**核心目的**：分析宏观环境因素，识别机会和威胁

**使用场景**：
- 进入新市场前
- 战略规划
- 风险评估
- 融资BP

## P - Political (政治因素)

**问题**：政治环境如何影响业务？

### 分析要点

| 维度 | 分析问题 | 影响 |
|-----|---------|------|
| **政策稳定性** | 政局稳定吗？政策会变吗？ | 政策风险 |
| **政府态度** | 政府支持/反对这个行业？ | 机会/威胁 |
| **贸易政策** | 有贸易壁垒/关税吗？ | 成本/市场 |
| **税收政策** | 税收如何变化？ | 利润 |
| **政府补贴** | 有政策支持/补贴吗？ | 成本 |
| **外交关系** | 与其他国家关系如何？ | 国际扩张 |

### 数据来源

- 政府工作报告
- 政策文件/通知
- 新闻报道
- 行业协会报告
- 咨询机构分析

## 从 PESTEL 到策略

### 机会识别

| 机会类型 | 来源 | 策略 |
|---------|------|------|
| **政策红利** | P | 快速进入 |
| **经济增长** | E | 扩大规模 |
| **社会趋势** | S | 产品创新 |
| **技术突破** | T | 技术升级 |
| **环保需求** | E | 绿色产品 |
| **法规变化** | L | 合规优势 |

### 风险预警

| 风险类型 | 来源 | 应对 |
|---------|------|------|
| **政策风险** | P | 政策监控/多元布局 |
| **经济波动** | E | 现金储备/弹性成本 |
| **社会变化** | S | 持续调研 |
| **技术颠覆** | T | 技术跟踪/研发投入 |
| **环境约束** | E | 绿色转型 |
| **法律风险** | L | 合规管理/法律顾问 |

### 进入决策框架

```
高机会
    │
    │      [快速进入]
    │      政策好、经济强、
    │      技术成熟、风险低
    │
    │  [谨慎进入]     [不考虑]
    │  有机会但有风险   机会少风险大
    │
    │  [观望]
    │  有潜力但不成熟
    └───────────────────── 低机会
         低风险
```

## PESTEL 与其他工具配合

### 配合使用

| 工具 | 用途 | 与 PESTEL 配合 |
|-----|------|---------------|
| **SWOT** | 综合评估 | PESTEL提供O/T输入 |
| **五力模型** | 行业竞争 | PESTEL是宏观背景 |
| **趋势分析** | 判断趋势 | PESTEL是趋势来源 |
| **TAM/SAM/SOM** | 市场规模 | PESTEL影响市场 |

### 使用顺序建议

```
1. PESTEL → 扫描宏观环境
2. 五力模型 → 分析行业竞争
3. TAM/SAM/SOM → 估算市场规模
4. SWOT → 综合评估，制定策略
```

## 常见错误

| 错误 | 问题 | 正确做法 |
|-----|------|---------|
| **分析太浅** | 只列因素不分析 | 深入到影响和策略 |
| **忽视动态** | 市场在变 | 定期更新 |
| **孤立分析** | 不结合具体业务 | 与目标市场/产品结合 |
| **只看威胁** | 过于悲观 | 平衡机会和威胁 |
| **分析后不行动** | 白做分析 | 转化为具体行动 |
| **数据来源不清** | 拍脑袋 | 标注数据来源 |

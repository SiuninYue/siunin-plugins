# SWOT 分析指南

## 模型概览

```
                内部因素                  外部因素
              ┌──────────────────────────────────┐
    积极      │                                    │
              │   Strengths (优势)    Opportunities (机会)
              │   ┌──────────┐         ┌──────────┐
              │   │ 内部优势 │         │ 外部机会 │
              │   │ 可以掌控 │         │ 可以利用 │
              │   └──────────┘         └──────────┘
              │
    消极      │   Weaknesses (劣势)   Threats (威胁)
              │   ┌──────────┐         ┌──────────┐
              │   │ 内部劣势 │         │ 外部威胁 │
              │   │ 需要改进 │         │ 需要规避 │
              │   └──────────┘         └──────────┘
              │                                    │
              └──────────────────────────────────┘
```

**核心目的**：系统评估内外部因素，制定匹配策略

**记忆口诀**：内看优劣势，外看机威胁

---

## SWOT 详解

### S - Strengths (优势)

**问题**：我们有什么内部优势？

| 分析维度 | 问题示例 |
|---------|---------|
| **产品** | 有什么独特功能？技术领先吗？ |
| **团队** | 核心能力是什么？有行业经验吗？ |
| **资源** | 有什么独家资源？资金充裕吗？ |
| **品牌** | 知名度如何？用户口碑怎样？ |
| **渠道** | 有特殊渠道吗？获客成本低吗？ |

**常见优势示例**：
- 技术壁垒 / 专利
- 独家数据 / 资源
- 优秀团队 / 行业经验
- 品牌认知 / 用户基础
- 资金优势 / 投资支持

---

### W - Weaknesses (劣势)

**问题**：我们有什么内部短板？

| 分析维度 | 问题示例 |
|---------|---------|
| **产品** | 功能缺失？质量不稳定？ |
| **团队** | 能力不足？人手不够？ |
| **资源** | 资金紧张？资源有限？ |
| **品牌** | 没知名度？负面评价？ |
| **运营** | 获客成本高？留存率低？ |

**常见劣势示例**：
- 产品未验证 / 功能不完善
- 团队经验不足
- 资金有限 / 烧钱快
- 无品牌认知
- 无用户基础

---

### O - Opportunities (机会)

**问题**：外部有什么机会可以利用？

| 分析维度 | 问题示例 |
|---------|---------|
| **市场** | 市场在增长吗？有新细分市场？ |
| **竞争** | 竞品有弱点吗？市场空白？ |
| **技术** | 新技术降低成本？新模式？ |
| **政策** | 政策支持？监管放松？ |
| **社会** | 用户习惯变化？新需求？ |

**常见机会示例**：
- 市场快速增长
- 竞品弱点 / 市场空白
- 新技术降低门槛
- 政策利好
- 用户习惯改变

---

### T - Threats (威胁)

**问题**：外部有什么威胁需要规避？

| 分析维度 | 问题示例 |
|---------|---------|
| **竞争** | 竞品强势？大厂进入？ |
| **替代** | 有替代方案？用户不需要？ |
| **政策** | 监管收紧？政策风险？ |
| **经济** | 经济下行？融资困难？ |
| **技术** | 技术迭代？被淘汰？ |

**常见威胁示例**：
- 巨头进入 / 竞争加剧
- 替代方案出现
- 监管政策变化
- 经济环境恶化
- 技术被颠覆

## SWOT 到策略

### 四种战略类型

| 战略类型 | 含义 | 适用情况 | 风险 |
|---------|------|---------|------|
| **SO 战略** | 发挥优势抓住机会 | 优势明显 + 机会很多 | 低 |
| **WO 战略** | 利用机会弥补劣势 | 劣势明显 + 机会很多 | 中 |
| **ST 战略** | 利用优势应对威胁 | 优势明显 + 威胁很多 | 中 |
| **WT 战略** | 减少劣势规避威胁 | 劣势明显 + 威胁很多 | 高 |

### 策略制定原则

**SO 战略（增长型）**：
- 最大化利用优势
- 全力抓住机会
- 快速扩张

**WO 战略（扭转型）**：
- 补短板以抓住机会
- 寻求合作 / 外包
- 渐进式改进

**ST 战略（多元化）**：
- 用优势对抗威胁
- 多元化布局
- 建立壁垒

**WT 战略（防御型）**：
- 收缩战线
- 砍掉非核心业务
- 保存实力

## 实用技巧

### 如何找出好的 SWOT

**优势分析**：
- 从用户视角看：用户为什么选我们？
- 从竞品视角看：竞品为什么学不会？
- 从数据视角看：什么指标表现好？

**劣势分析**：
- 从用户投诉看：用户抱怨什么？
- 从流失分析看：用户为什么走？
- 从团队痛点看：什么总是做不好？

**机会分析**：
- 看趋势：什么在增长？
- 看竞品：竞品哪里做得不好？
- 看政策：有什么红利？

**威胁分析**：
- 看大厂：他们会不会进来？
- 看替代：有没有替代方案？
- 看变化：技术/政策会不会变？

## 常见错误

| 错误 | 问题 | 正确做法 |
|-----|------|---------|
| **混淆内外** | 把外部威胁写成内部劣势 | 分清内外因素 |
| **过于笼统** | "团队很棒" 但没有具体内容 | 具体到可衡量 |
| **只列不做** | 分析完就完了 | 转化为具体行动 |
| **过度乐观** | 只看好的不看坏的 | 客观评估 |
| **一次分析** | 市场在变，分析要更新 | 定期复盘 |
| **孤立分析** | 不结合具体场景 | 与目标市场/阶段结合 |

## SWOT 与其他工具配合

### 配合使用

| 工具 | 用途 | 与 SWOT 配合 |
|-----|------|-------------|
| **TAM/SAM/SOM** | 评估市场规模 | O 市场机会的数据支撑 |
| **竞品分析** | 了解竞争对手 | S/W/T 的具体内容 |
| **五力模型** | 评估行业竞争 | T 的行业分析 |
| **PESTEL** | 分析宏观环境 | O/T 的外部因素 |

### 使用顺序建议

```
1. TAM/SAM/SOM → 市场有多大 (O)
2. 竞品分析 → 竞争对手怎么样 (T)
3. PESTEL → 宏观环境如何 (O/T)
4. SWOT → 综合评估，制定策略
```

## 快速检查清单

做 SWOT 分析前，确保：

- [ ] 明确分析目的（为什么做这个分析）
- [ ] 定义分析对象（产品/项目/公司）
- [ ] 收集必要信息（市场、竞品、内部）
- [ ] 区分内外因素
- [ ] 具体、可衡量
- [ ] 转化为行动

分析后，确保：

- [ ] 有明确的策略建议
- [ ] 有优先级行动计划
- [ ] 有时间表
- [ ] 有责任人

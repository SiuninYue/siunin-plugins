# Porter 五力模型

## 模型概览

```
                    新进入者威胁
                         │
                         ▼
     供应商 ─────────► 行业竞争 ◄─────── 买家
     议价能力           程度           议价能力
                         │
                         ▼
                    替代品威胁
```

**核心目的**：分析行业的盈利能力和竞争激烈程度

**记忆口诀**：上下左右中 = 新进入者、替代品、供应商、买家、行业竞争

---

## 五力详解

### 1. 新进入者威胁

**问题**：新玩家进入这个市场有多难？

| 因素 | 进入壁垒高 | 进入壁垒低 |
|-----|----------|----------|
| **资本需求** | 需要大量资金 | 低成本启动 |
| **技术门槛** | 需要专利/专业技术 | 技术可复制 |
| **规模效应** | 规模越大成本越低 | 规模不重要 |
| **渠道关系** | 渠道排他 | 渠道开放 |
| **政策限制** | 需要牌照/审批 | 无准入限制 |
| **品牌效应** | 用户忠诚度高 | 用户容易切换 |

**分析模板**：
```markdown
## 新进入者威胁分析

**威胁程度**：高 / 中 / 低

**主要进入壁垒**：
- [壁垒1]：强度 高/中/低
- [壁垒2]：强度 高/中/低

**潜在进入者**：
- [谁可能进入]
- [他们的优势是什么]

**应对策略**：
- [如何提高壁垒]
```

---

### 2. 替代品威胁

**问题**：用户不用你的产品，还能用什么？

| 因素 | 替代威胁高 | 替代威胁低 |
|-----|----------|----------|
| **替代品数量** | 选择很多 | 几乎无替代 |
| **切换成本** | 切换容易 | 切换代价大 |
| **性价比** | 替代品更便宜 | 替代品更贵 |
| **用户习惯** | 用户愿意尝试 | 用户习惯固定 |

**替代品类型**：
- **直接替代**：功能相同的其他产品
- **间接替代**：用不同方式解决同一问题
- **不消费**：用户选择不解决这个问题

**分析模板**：
```markdown
## 替代品威胁分析

**威胁程度**：高 / 中 / 低

**主要替代品**：
| 替代品 | 类型 | 优势 | 劣势 |
|-------|------|------|------|
| [替代品1] | 直接/间接 | [优势] | [劣势] |
| [替代品2] | 直接/间接 | [优势] | [劣势] |

**用户切换动机**：
- [什么情况下用户会切换]

**防御策略**：
- [如何降低替代威胁]
```

### 3. 买家议价能力

**问题**：客户能压价吗？

| 因素 | 买家强势 | 卖家强势 |
|-----|---------|---------|
| **买家集中度** | 少数大客户 | 客户分散 |
| **采购规模** | 单客户采购量大 | 单客户采购少 |
| **切换成本** | 容易换供应商 | 换供应商代价大 |
| **信息透明度** | 价格透明 | 价格不透明 |
| **产品差异化** | 产品同质化 | 产品有独特价值 |
| **后向整合** | 买家可以自己做 | 买家做不了 |

**分析模板**：
```markdown
## 买家议价能力分析

**买家议价能力**：强 / 中 / 弱

**主要客户群体**：
| 客户类型 | 占比 | 议价能力 | 切换成本 |
|---------|------|---------|---------|
| [类型1] | X% | 高/中/低 | 高/中/低 |
| [类型2] | X% | 高/中/低 | 高/中/低 |

**议价压力来源**：
- [压力1]
- [压力2]

**应对策略**：
- [如何降低买家议价能力]
```

### 4. 供应商议价能力

**问题**：供应商能涨价吗？

| 因素 | 供应商强势 | 买家强势 |
|-----|----------|---------|
| **供应商集中度** | 供应商少 | 供应商多 |
| **替代供应商** | 难以替换 | 容易替换 |
| **切换成本** | 换供应商代价大 | 换供应商容易 |
| **供应商专业性** | 提供独特资源 | 提供通用资源 |
| **前向整合** | 供应商可能直接卖 | 供应商不会绕过你 |

**分析模板**：
```markdown
## 供应商议价能力分析

**供应商议价能力**：强 / 中 / 弱

**主要供应商**：
| 供应商类型 | 依赖程度 | 替代难度 | 议价能力 |
|-----------|---------|---------|---------|
| [类型1] | 高/中/低 | 高/中/低 | 强/中/弱 |
| [类型2] | 高/中/低 | 高/中/低 | 强/中/弱 |

**关键供应商风险**：
- [风险1]
- [风险2]

**应对策略**：
- [如何降低供应商依赖]
```

### 5. 行业竞争程度

**问题**：现有玩家之间打得多激烈？

| 因素 | 竞争激烈 | 竞争温和 |
|-----|---------|---------|
| **竞争者数量** | 玩家很多 | 玩家很少 |
| **行业增速** | 增长慢/萎缩 | 快速增长 |
| **产品差异** | 产品同质化 | 差异化明显 |
| **固定成本** | 固定成本高 | 固定成本低 |
| **退出壁垒** | 退出代价大 | 容易退出 |
| **战略多样性** | 战略相似 | 战略多样 |

**分析模板**：
```markdown

**竞争激烈程度**：高 / 中 / 低

**主要竞争者**：
| 竞争者 | 市场份额 | 定位 | 竞争策略 |
|-------|---------|------|---------|
| [竞争者1] | X% | [定位] | [策略] |
| [竞争者2] | X% | [定位] | [策略] |

**竞争焦点**：
- [价格战/功能战/服务战/...]

**竞争趋势**：
- [竞争是否在加剧]

**差异化机会**：
- [我们可以如何差异化]
```

## 五力模型局限性

| 局限 | 说明 |
|-----|------|
| **静态分析** | 只看当前，不看变化 |
| **忽视互补品** | 只考虑替代品 |
| **假设对立** | 实际可能合作共赢 |
| **行业边界模糊** | 跨界竞争越来越多 |

**使用建议**：五力分析是起点，不是终点。结合动态趋势和具体情况综合判断。

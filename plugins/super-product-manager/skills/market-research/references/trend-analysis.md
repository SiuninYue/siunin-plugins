## 趋势判断框架

### 五维评估法

| 维度 | 问题 | 评估标准 |
|-----|------|---------|
| **真实性** | 这是真趋势还是炒作？ | 有实际用户/收入支撑 |
| **时机** | 是太早还是太晚？ | 基础设施成熟度 |
| **规模** | 市场有多大？ | TAM/SAM 估算 |
| **持续性** | 能持续多久？ | 结构性变化 vs 短期热点 |
| **相关性** | 和我有什么关系？ | 能力匹配度 |

### 趋势真伪判断

| 信号 | 真趋势 | 假趋势/炒作 |
|-----|-------|-----------|
| **用户行为** | 用户习惯真的在变 | 只有媒体在说 |
| **资金流向** | 大量资金持续投入 | 只有几个案例 |
| **基础设施** | 支撑技术已成熟 | 核心技术还不成熟 |
| **盈利模式** | 有公司开始赚钱 | 只有估值没有收入 |
| **时间跨度** | 持续 2-3 年以上 | 热了几个月就冷了 |

### Gartner 技术成熟度曲线

```
期望值
  │
  │        ╭─╮ 期望膨胀期
  │       ╱   ╲
  │      ╱     ╲
  │     ╱       ╲        ╱────── 生产成熟期
  │    ╱         ╲      ╱
  │   ╱           ╲────╱ 复苏期
  │  ╱             ╲
  │ ╱               ╲ 幻灭期
  │╱  技术萌芽期     ╲
  └──────────────────────────────── 时间
```

| 阶段 | 特点 | 策略 |
|-----|------|------|
| **技术萌芽期** | 概念出现，媒体关注 | 观望，研究 |
| **期望膨胀期** | 大量资金涌入，估值飙升 | 谨慎，警惕泡沫 |
| **幻灭期** | 泡沫破裂，批评声起 | 布局，低价进入 |
| **复苏期** | 少数公司跑出来 | 跟进，学习模式 |
| **生产成熟期** | 商业化成功，广泛应用 | 规模化，竞争加剧 |

## 常见错误

| 错误 | 问题 | 正确做法 |
|-----|------|---------|
| **跟风太快** | 趋势还不成熟就冲进去 | 评估时机 |
| **跟风太慢** | 等趋势成熟再进入，已经太晚 | 早期布局 |
| **只看正面** | 只收集支持进入的信息 | 寻找反面证据 |
| **忽视执行** | 趋势对但执行跟不上 | 评估自身能力 |
| **分析瘫痪** | 分析太久不做决策 | 设定决策时间 |

## 趋势与战略的连接

### 从趋势到战略

```
识别趋势 → 评估机会 → 能力匹配 → 资源配置 → 执行计划
```

### 战略响应模式

| 模式 | 适用场景 | 风险 |
|-----|---------|------|
| **引领者** | 有技术/资源优势 | 高风险高回报 |
| **快速跟随者** | 验证后快速跟进 | 中等风险 |
| **后进者** | 等市场成熟再进入 | 低风险低回报 |
| **观望者** | 趋势与主业关系不大 | 可能错过机会 |

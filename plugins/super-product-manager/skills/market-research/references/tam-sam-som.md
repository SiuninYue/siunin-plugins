# TAM/SAM/SOM 市场规模估算

## 三层市场模型

```
┌─────────────────────────────────────────────┐
│                    TAM                       │
│            Total Addressable Market          │
│               (理论最大市场)                  │
```

## 定义详解

| 层级 | 英文全称 | 定义 | 计算方式 |
|-----|---------|------|---------|
| **TAM** | Total Addressable Market | 如果你占领 100% 市场，能赚多少 | 行业总规模 |
| **SAM** | Serviceable Addressable Market | 你的产品形态能覆盖的市场 | TAM × 目标用户比例 |
| **SOM** | Serviceable Obtainable Market | 你实际能拿到的市场 | SAM × 预期市场份额 |

### 形象比喻

- **TAM**：整个蛋糕有多大
- **SAM**：你能切到的那块蛋糕
- **SOM**：你实际能吃到的那口蛋糕

---

## 估算方法

### 方法一：自上而下 (Top-Down)

**原理**：从行业报告出发，逐层收窄

```
TAM = 行业报告数据
SAM = TAM × 目标细分市场比例
SOM = SAM × 预期市场份额 (新进入者通常 1-5%)
```

**示例：企业协作工具**
```markdown
TAM = 全球企业软件市场 = 5000 亿美元（Gartner 报告）
SAM = 协作工具细分市场 = 5000 × 3% = 150 亿美元
SOM = 中国中小企业市场，预期份额 2% = 150 × 10% × 2% = 3000 万美元
```

**优点**：简单快速，有权威来源
**缺点**：可能过于乐观，假设较多

### 方法二：自下而上 (Bottom-Up)

**原理**：从单个用户/交易出发，逐步累加

```
单用户价值 = 客单价 × 购买频率
目标用户数 = 可触达用户数
市场规模 = 单用户价值 × 目标用户数
```

**示例：SaaS 工具**
```markdown
单用户年价值 = ¥1200/年
目标用户数 = 中国 500 万家中小企业 × 10% 有需求 = 50 万家
SOM = ¥1200 × 50 万 × 5% 渗透率 = ¥3 亿/年
```

**优点**：更贴近实际，假设可验证
**缺点**：需要更多市场数据

### 方法三：价值理论 (Value Theory)

**原理**：基于用户获得的价值计算

```
市场规模 = 用户痛点成本 × 用户数量 × 愿意支付比例
```

**示例：效率工具**
```markdown
用户时间成本 = ¥500/月（浪费在低效工作上）
用户数量 = 1000 万知识工作者
愿意支付 = 节省成本的 20%
市场规模 = ¥500 × 20% × 1000 万 × 12 个月 = ¥120 亿/年
```

## 数据来源

### 行业报告来源

| 来源 | 类型 | 特点 |
|-----|------|------|
| **艾瑞咨询** | 互联网/消费 | 中国市场，免费部分可用 |
| **QuestMobile** | 移动互联网 | 月活、时长数据 |
| **IDC** | 企业IT | 全球视角，付费 |
| **Gartner** | 企业软件 | 权威，付费 |
| **Statista** | 综合 | 图表丰富，部分免费 |
| **国家统计局** | 宏观经济 | 官方数据 |

### 辅助验证方法

| 方法 | 做法 |
|-----|------|
| **竞品财报** | 上市竞品的收入 × 市场份额倒推 |
| **招聘规模** | 竞品招聘人数推算业务规模 |
| **App 下载量** | 从下载量推算用户规模 |
| **搜索指数** | 百度指数看需求趋势 |

## 常见错误

| 错误 | 问题 | 正确做法 |
|-----|------|---------|
| **只看 TAM** | TAM 再大跟你没关系 | 重点关注 SOM |
| **假设过于乐观** | "我们只需要 1%..." | 1% 也很难拿到 |
| **忽视竞争** | 市场大不代表你能拿到 | 分析竞争格局 |
| **数据来源不清** | 拍脑袋的数字 | 标注所有来源 |
| **静态视角** | 只看现在不看趋势 | 考虑市场增长率 |

## SOM 现实检验

### 新进入者的真实份额

| 市场类型 | 第一年 SOM | 说明 |
|---------|----------|------|
| 新兴市场 | 5-10% | 竞争少，增长快 |
| 成熟市场 | 1-3% | 竞争激烈，需要差异化 |
| 垄断市场 | <1% | 巨头主导，很难撼动 |

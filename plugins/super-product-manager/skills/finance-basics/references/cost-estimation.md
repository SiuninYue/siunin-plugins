# 成本估算框架

## 成本分类

### 按性质分

| 类型 | 说明 | 示例 |
|------|------|------|
| **固定成本** | 不随业务量变化 | 服务器基础费、SaaS 订阅 |
| **可变成本** | 随业务量变化 | API 调用费、流量费 |
| **沉没成本** | 已发生不可回收 | 已投入的开发成本 |
| **机会成本** | 放弃其他选择的代价 | 做A就不能做B |

### 按阶段分

| 阶段 | 成本项 |
|------|--------|
| **开发期** | 人力、外包、采购 |
| **上线期** | 服务器、推广、培训 |
| **运营期** | 维护、客服、迭代 |

## 人力成本估算

### 日均成本计算

```
日均成本 = (年薪 + 五险一金 + 福利) ÷ 250 工作日

示例：
- 初级开发：¥20万/年 → ¥800/天
- 中级开发：¥35万/年 → ¥1,400/天
- 高级开发：¥50万/年 → ¥2,000/天
- 产品经理：¥30万/年 → ¥1,200/天
- 设计师：¥25万/年 → ¥1,000/天
```

### PERT 估算法

```
预估工时 = (乐观 + 4×正常 + 悲观) ÷ 6

示例：
乐观 = 5天，正常 = 8天，悲观 = 15天
预估 = (5 + 4×8 + 15) ÷ 6 = 8.7天 ≈ 9天
```

## 常见低估点

1. **沟通成本**：会议、评审、协调时间
2. **返工成本**：需求变更、Bug 修复
3. **运维成本**：监控、值班、故障处理
4. **迭代成本**：上线后持续优化
5. **隐性成本**：招聘、培训、管理开销

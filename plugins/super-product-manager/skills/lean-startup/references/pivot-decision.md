# Pivot or Persevere 决策

## 什么时候该问这个问题

**定期问自己**：我们应该继续（Persevere）还是转型（Pivot）？

建议频率：
- 早期：每 2-4 周
- 成长期：每季度
- 成熟期：每半年

---

## 转型信号

### 需要考虑转型的信号

| 信号 | 说明 | 严重程度 |
|-----|------|---------|
| **核心指标不增长** | 北极星指标持续 4+ 周无增长 | ⭐⭐⭐ |
| **留存极低** | 周留存 < 10% | ⭐⭐⭐ |
| **没人愿意付钱** | 测试付费无人买单 | ⭐⭐⭐ |
| **增长只靠烧钱** | 停止投放就无增长 | ⭐⭐ |
| **用户反馈负面** | 核心功能差评多 | ⭐⭐ |
| **团队士气低落** | 大家都不相信了 | ⭐⭐ |

### 不需要转型的假信号

| 假信号 | 实际情况 |
|-------|---------|
| 短期波动 | 正常波动，需要更长时间观察 |
| 竞品动作 | 竞品做什么不代表你要跟 |
| 投资人意见 | 投资人不一定比你更懂用户 |
| 新闻热点 | 热点不等于机会 |

## 决策框架

### 三个核心问题

```
1. 数据怎么说？
   - 核心指标趋势
   - 用户行为数据
   - 商业数据

2. 用户怎么说？
   - 定性反馈
   - 用户访谈
   - 流失原因

3. 为什么会这样？
   - 问题层面：问题不存在？
   - 方案层面：方案不对？
   - 执行层面：执行有问题？
```

### 决策树

```
┌─────────────────────────────────────────┐
│           数据怎么说？                   │
│                                         │
│  核心指标在增长？                         │
│     │                                   │
│     ├─ YES → 继续，加大投入              │
│     │                                   │
│     └─ NO → 为什么？                    │
│             │                           │
│             ├─ 假设错了 → 转型           │
│             │                           │
│             ├─ 执行问题 → 改进执行        │
│             │                           │
│             └─ 市场问题 → 考虑放弃        │
│                                         │
└─────────────────────────────────────────┘
```

## 转型类型

### 10 种转型方式

| 类型 | 描述 | 例子 |
|-----|------|------|
| **用户细分转型** | 换一群用户 | B2C → B2B |
| **问题转型** | 换一个问题 | 发现真正痛点在别处 |
| **解决方案转型** | 换一种方案 | App → 小程序 |
| **渠道转型** | 换获客渠道 | 线上 → 线下 |
| **收入模式转型** | 换收费方式 | 订阅 → 买断 |
| **引擎转型** | 换增长引擎 | 付费 → 病毒 |
| **技术转型** | 换技术方案 | 人工 → AI |
| **平台转型** | 从产品到平台 | 单一产品 → 平台生态 |
| **价值捕获转型** | 换变现对象 | 用户付费 → 广告 |
| **缩放转型** | 调整产品范围 | 大而全 → 小而精 |

## 转型 ≠ 失败

### 著名的成功转型

| 公司 | 转型前 | 转型后 |
|-----|-------|-------|
| **Slack** | 游戏公司 | 企业通讯工具 |
| **Instagram** | 位置签到 App | 图片分享 |
| **YouTube** | 视频约会网站 | 视频分享平台 |
| **Twitter** | 播客平台 | 微博客 |
| **Nintendo** | 扑克牌公司 | 游戏公司 |

### 转型心态

**转型不是**：
- ❌ 承认失败
- ❌ 放弃原来的努力
- ❌ 从头开始

**转型是**：
- ✅ 基于学习调整方向
- ✅ 保留有价值的部分
- ✅ 更聪明地前进

## 什么时候该放弃

### 放弃信号

| 信号 | 说明 |
|-----|------|
| **市场不存在** | 验证后确认没有需求 |
| **没有差异化** | 做不到比现有方案更好 |
| **资源耗尽** | 没有时间和资金继续 |
| **团队分崩** | 核心成员离开 |
| **多次转型失败** | 尝试多个方向都不行 |

### 放弃不是失败

**学到了**：
- 什么不 work
- 市场的真实情况
- 团队的能力边界
- 创业的经验

**可以带走的**：
- 技术积累
- 团队关系
- 行业认知
- 经验教训

# 行业资质要求

## 基础资质（通用）

| 资质 | 适用场景 | 办理要求 |
|------|---------|---------|
| **ICP 备案** | 境内网站 | 企业：营业执照；个人：身份证 |
| **ICP 许可证** | 经营性网站 | 注册资金100万+，3人社保 |
| **EDI 许可证** | 在线数据处理 | 电商平台、数据交易 |
| **网络文化经营许可** | 游戏、音视频 | 文化部门审批 |

## 行业专项资质

### 金融类

```markdown
## 金融资质矩阵

| 业务类型 | 所需资质 | 监管部门 | 难度 |
|---------|---------|---------|------|
| 第三方支付 | 支付业务许可证 | 央行 | 极难 |
| 网络借贷 | 金融牌照 | 银保监 | 停止新批 |
| 基金销售 | 基金销售牌照 | 证监会 | 难 |
| 保险销售 | 保险经纪/代理牌照 | 银保监 | 中 |
| 证券咨询 | 证券投资咨询牌照 | 证监会 | 难 |

### 无牌照替代方案
| 业务 | 替代方案 |
|------|---------|
| 支付 | 接入持牌支付公司 |
| 借贷 | 与银行合作助贷 |
| 保险 | 申请保险经纪牌照 |
```

## 资质办理流程

### 通用流程

```markdown
## 资质办理标准流程

### 1. 评估阶段
- 确定业务范围
- 匹配所需资质
- 评估办理难度和周期

### 2. 准备阶段
- 注册公司（部分资质有地区要求）
- 准备材料（营业执照、场地、人员）
- 完成前置条件（如等保）

### 3. 申请阶段
- 提交申请材料
- 等待审批
- 补充材料（如需）

### 4. 维护阶段
- 年检/年报
- 变更备案
- 续期办理
```

## 合规策略建议

### 无资质时的合规方案

| 场景 | 替代方案 |
|------|---------|
| 需要支付资质 | 接入第三方支付 |
| 需要金融资质 | 与持牌机构合作 |
| 需要视听资质 | 接入有资质平台 |
| 需要游戏版号 | 代理已有版号游戏 |
| 需要医疗资质 | 与医院合作 |

### 风险等级评估

| 风险等级 | 情况 | 建议 |
|---------|------|------|
| 高 | 无资质开展需资质业务 | 立即停止或获取资质 |
| 中 | 资质范围模糊地带 | 主动咨询监管部门 |
| 低 | 确定不需要资质 | 持续关注政策变化 |

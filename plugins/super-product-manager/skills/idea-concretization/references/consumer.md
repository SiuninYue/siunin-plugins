# 消费品/食品

## 核心价值主张

- **用户价值**：为目标消费者提供[独特功能/情感体验/便利性]，在[使用场景]下解决[具体痛点]
- **商业价值**：通过[零售/批发/订阅]模式，实现[毛利率目标]和[市场规模目标]，差异化在于[产品特性/品牌故事/用户体验]
- **验证指标**：复购率 > [X]%，NPS > [Y]，单次获客成本 < [Z]元

## 追加问题

1. **目标用户与场景**：具体是哪些人群在什么情况下使用？购买决策的关键因素是什么？
2. **产品差异化**：与现有竞品相比，在功能、设计、价格、体验上的核心优势是什么？
3. **供应链稳定性**：原料供应是否稳定？生产周期和最小起订量是多少？质量控制标准如何？
4. **渠道策略**：线上（电商平台、自营网站）和线下（零售店、经销商）的布局和优先级？
5. **定价与毛利**：零售价、成本结构、渠道分成后的到手毛利是多少？价格测试结果如何？
6. **品牌建设**：品牌故事、视觉识别、包装设计如何传达核心价值？营销预算分配？
7. **合规与安全**：需要哪些许可证、检测报告、标签要求？保质期和存储条件如何？

## 关键成功因素

- **产品力**：独特卖点清晰，用户体验优秀，品质稳定可靠
- **渠道力**：线上线下渠道布局合理，动销率高，库存周转快
- **品牌力**：品牌故事有感染力，视觉识别统一，用户口碑良好
- **供应链力**：供应稳定，成本可控，质量一致，可规模化生产
- **数据力**：销售数据、用户反馈、市场趋势的实时监控和分析

## 常见陷阱与规避

- **产品同质化**：缺乏独特卖点 → 先做小规模用户测试验证差异化
- **渠道依赖过重**：单一渠道占比过高 → 多渠道布局，降低依赖风险
- **库存积压**：销售预测不准 → 小批量试产，根据动销数据调整
- **品质不稳定**：供应链管理不足 → 建立严格的质量标准和供应商审核
- **定价不合理**：毛利空间不足 → 做价格敏感度测试，优化成本结构

## 建议文档（按需）

- **产品规格文档**：详细的产品功能、材质、尺寸、包装要求
- **供应链管理计划**：供应商清单、采购流程、质量控制标准、应急预案
- **渠道策略文档**：渠道选择标准、合作条款、动销目标、库存管理
- **品牌手册**：品牌定位、视觉识别系统、包装设计规范、品牌故事
- **财务模型**：成本结构分析、定价策略、损益预测、现金流管理
- **合规清单**：所需许可证、检测报告、标签要求、安全标准

## 验证优先级指导

- **P0（必须验证）**：目标用户真实需求、产品差异化接受度、价格敏感度
- **P1（重要验证）**：供应链可行性、渠道合作意愿、最小可行产能
- **P2（后续验证）**：品牌认知建立、用户复购行为、规模化生产成本

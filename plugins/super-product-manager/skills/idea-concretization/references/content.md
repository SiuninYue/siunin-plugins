# 内容策划（产品导向）

## 核心价值主张

- **用户价值**：为目标受众提供[有价值信息/情感共鸣/实用指导]，在[场景]下解决[信息需求/认知提升/决策支持]，建立信任/影响力/社群
- **商业价值**：通过[品牌建设/用户获取/直接变现]支持产品增长，实现[流量/转化/收入]目标，差异化在于[内容深度/独特视角/表达风格]
- **验证指标**：内容互动率 > [X]%，受众增长 > [Y]%/月，内容转化率 > [Z]%

## 追加问题

1. **内容目标**：内容策划的核心目标是什么（品牌认知、用户教育、需求生成、社区建设）？
2. **目标受众**：具体服务哪些人群？他们的信息需求、内容偏好、消费场景是什么？
3. **内容策略**：内容主题、形式（文字、视频、音频）、风格、发布频率如何规划？
4. **生产流程**：内容创作、编辑、审核、发布、优化的流程和资源需求？
5. **渠道分布**：内容发布渠道（自有平台、社交媒体、合作伙伴）和分发策略？
6. **效果衡量**：关键内容指标（阅读量、互动率、分享数、转化率）如何定义和追踪？
7. **持续优化**：如何基于数据反馈调整内容策略？A/B测试和用户反馈机制？

## 关键成功因素

- **内容价值明确**：提供独特、实用、有深度的信息价值
- **受众理解深刻**：精准把握目标受众的需求、痛点和内容偏好
- **生产流程高效**：内容创作、审核、发布流程标准化、可持续
- **数据驱动优化**：基于数据反馈持续改进内容策略和效果
- **与产品协同**：内容策略与产品目标、用户旅程紧密结合

## 常见陷阱与规避

- **内容同质化**：缺乏独特视角和价值 → 聚焦细分领域，建立专业权威
- **生产不可持续**：过度依赖个人创意 → 建立内容框架和流程标准化
- **效果无法衡量**：只关注产出不关注效果 → 建立完整的数据指标体系
- **与产品脱节**：内容与产品价值主张不一致 → 确保内容策略支持产品目标
- **受众增长停滞**：内容吸引力不足 → 持续测试新形式、新主题、新渠道

## 建议文档（按需）

- **产品内容策略**：定位、主题规划、价值主张、成功指标
- **目标受众分析**：用户画像、需求痛点、内容偏好、消费场景
- **内容生产流程**：创作、审核、发布、优化机制和工具
- **效果评估体系**：指标定义、数据收集、分析报告、优化迭代
- **内容日历与计划**：发布时间表、主题规划、资源分配、协作流程
- **品牌内容指南**：语调风格、视觉规范、质量标尺、合规要求

## 验证优先级指导

- **P0（必须验证）**：目标受众内容需求、核心内容价值、内容形式接受度
- **P1（重要验证）**：内容生产可行性、渠道分发效果、关键指标达成
- **P2（后续验证）**：内容规模化能力、品牌影响力建设、商业变现潜力

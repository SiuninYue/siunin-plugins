## 模板使用说明

### 什么时候使用这个模板

**数据PRD适用于**：
- 数据埋点需求
- 数据分析需求
- 数据看板需求
- 数据指标体系设计

**不适用的场景**：
- 纯功能需求 → 使用功能PRD
- 纯API对接 → 使用API PRD

### 使用技巧

1. **从业务问题出发**：明确数据要解决什么业务问题
2. **指标要可量化**：每个指标都要有明确的计算公式
3. **埋点要验证**：埋点开发后必须验证准确性
4. **考虑数据质量**：提前规划数据采集、处理、存储
5. **关注数据隐私**：确保符合隐私保护要求

## 模板使用说明

### 什么时候使用这个模板

**通用PRD适用于**：
- 综合性项目需求
- 包含多种类型的需求
- 不确定具体类型的项目

**与其他模板的选择**：
- 偏向用户功能 → 使用功能PRD
- 偏向数据分析 → 使用数据PRD
- 偏向API对接 → 使用API PRD

### 使用技巧

1. **灵活调整**：根据项目特点调整章节
2. **突出重点**：重要内容详细描述
3. **保持简洁**：避免冗余信息
4. **及时更新**：变更时更新版本和记录
5. **多方对齐**：与设计、开发、测试达成共识

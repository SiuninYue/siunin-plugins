## 协作沟通技巧

### 如何让对方愿意帮忙

| 技巧 | 做法 |
|-----|------|
| **说明收益** | 告诉对方能得到什么 |
| **降低成本** | 尽量减少对方的工作量 |
| **提前沟通** | 给对方足够的准备时间 |
| **提供支持** | 告诉对方你能提供什么帮助 |
| **尊重流程** | 走正式流程，不要总是"特事特办" |
| **及时反馈** | 让对方知道他的帮助产生了价值 |

### 跨团队沟通原则

| 原则 | 说明 |
|-----|------|
| **对事不对人** | 讨论问题，不攻击人 |
| **书面确认** | 重要事项书面记录 |
| **及时同步** | 有变化第一时间通知 |
| **换位思考** | 理解对方的难处 |
| **共同目标** | 强调共同的业务目标 |

## 处理协作冲突

### 常见冲突类型

| 类型 | 表现 | 处理方式 |
|-----|------|---------|
| **优先级冲突** | 对方说没空 | 升级决策，明确优先级 |
| **责任边界** | 谁该做这个事 | 明确职责，书面确认 |
| **进度延误** | 对方没按时交付 | 了解原因，调整计划 |
| **质量问题** | 交付物不符合预期 | 明确标准，返工 |

### 冲突升级路径

```
私下沟通
    │
    ▼ 无法解决
双方负责人沟通
    │
    ▼ 无法解决
共同上级决策
```

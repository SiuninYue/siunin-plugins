# Launch Templates

## 上线沟通模板

### 内部沟通

```markdown
# 上线通知：[功能名称]

## 上线时间
[日期] [时间]

## 上线内容
- 功能1：...
- 功能2：...

## 影响范围
- 影响：[说明]
- 不影响：[说明]

## 注意事项
- [ ] 上线期间可能会有短暂服务中断
- [ ] 上线后请关注[监控指标]

## 值班安排
- 技术值班：[姓名] [联系方式]
- 产品值班：[姓名] [联系方式]
```

### 外部沟通（用户公告）

```markdown
# 新功能上线通知

亲爱的用户：

我们很高兴地宣布，[功能名称]正式上线啦！

## 新功能介绍
[简洁说明功能价值]

## 如何使用
[使用指南]

## 常见问题
[FAQ]

如有问题，请联系客服：[联系方式]
```

## 上线内容
1. [功能1]
2. [功能2]

- [ ] 延期完成（原因：...）
- [ ] 回滚（原因：...）

## 数据表现

|-----|-------|-------|-----|
| DAU | ... | ... | ... |

| 响应时间 | ... | ... | ... |

## 遇到的问题
1. [问题描述] - [解决方案]
2. [问题描述] - [解决方案]

## 后续计划
